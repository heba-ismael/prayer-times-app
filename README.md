# Prayer Times App

A modern React application that provides accurate prayer times with a clean and responsive UI.

## Features
- 🕌 Accurate prayer times
- 📍 Auto location detection (GPS)
- 🧭 Qibla direction
- ⏳ Next prayer countdown
- 🌍 Arabic & English support
- 📅 Weekly prayer schedule
- 📱 Responsive design

## Tech Stack
- React
- Vite
- Fetch API
- CSS
- Aladhan API
